About Smart Contact Manager

Smart Contact Manager is a modern web-based application built using Spring Boot that allows users to manage their contacts easily and securely with a clean UI and mobile-responsive design.

Add & Manage Contacts

Organize all your contacts in one place with full control to add, edit, or delete.

Secure & Private

Your contacts are protected with authentication, role-based access, and encryption.

Mobile Responsive

Accessible on mobile, tablet, and desktop using Bootstrap 4 layout.
